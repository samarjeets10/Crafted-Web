# Agency.ai

A modern agency website focused on creating a premium browsing experience through thoughtful interaction design, motion, and frontend architecture.

Built with React, Vite, Tailwind CSS, and Framer Motion.

---

## Overview

Agency.ai started as an exploration into how motion and interaction design can elevate a modern web experience beyond static layouts.

The objective was not simply to build another landing page, but to create an interface that feels responsive, intentional, and polished at every touchpoint.

Throughout the project, particular attention was given to animation timing, navigation flow, visual hierarchy, theme persistence, component reusability, and responsive behavior. Every interaction—from hover states to page transitions—was designed to contribute to a cohesive user experience rather than serve as visual decoration.

The project also became an opportunity to explore Framer Motion in depth and understand how motion can be integrated into React applications while maintaining performance and usability.

## Core Highlights

### Motion-Driven User Experience

Framer Motion is used throughout the application to introduce subtle movement, section reveals, hover interactions, and scroll-based transitions.

The animations are intentionally restrained and functional, guiding user attention without overwhelming the interface.

### Theme Architecture

The application supports both light and dark themes with persistent user preferences.

Theme selection is stored using localStorage and restored during application initialization to ensure a consistent experience across sessions.

### Component System

The UI is built around reusable React components with a focus on maintainability and scalability.

Shared functionality is abstracted where appropriate, reducing duplication and improving long-term project structure.

### Navigation Experience

The website uses smooth section-based navigation to create a seamless browsing experience.

Links are designed to guide users naturally through the content while maintaining orientation throughout the page.

### Responsive Design

Every section has been optimized for a wide range of screen sizes using Tailwind CSS.

Layouts, spacing systems, typography scales, and interaction patterns adapt fluidly across desktop, tablet, and mobile devices.

### Custom Cursor System

Instead of relying on the default browser cursor, the project introduces a custom animated cursor with contextual interaction handling.

The implementation enhances the visual identity of the website while preserving usability.

### Contact Workflow

The contact section integrates Web3Forms to enable direct form submissions without requiring a custom backend.

Messages are delivered directly to the configured email address while maintaining a streamlined user experience.

* State lifting and shared state management

* React Hooks (useState, useEffect)
* Framer Motion animation orchestration
* Responsive design systems
* User-centered interaction design
* Scroll-based experience design
* Progressive visual feedback
* Reusable UI patterns
